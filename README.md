# QA Training App - Automation Suite

**Group 31 - Cucumber + Playwright**

## Project Setup

### Prerequisites

- Node.js v20+
- Java JDK 21
- MySQL 8.0
- QA Training App running on localhost:8080

### Installation

```bash
cd automation-suite
npm install
npx playwright install
```

### Running Tests

```bash
# Run all UI tests
npm run test:ui

# Run all API tests
npm run test:api

# Run all tests
npm test
```

## Project Structure

```
automation-suite/
├── src/test/
│   ├── features/          # Gherkin feature files
│   ├── step-definitions/  # Step implementation
│   ├── pages/            # Page Object Models
│   ├── support/          # Hooks and world
│   └── utils/            # Helper functions
├── reports/              # Test reports
└── cucumber.js           # Cucumber config
```
